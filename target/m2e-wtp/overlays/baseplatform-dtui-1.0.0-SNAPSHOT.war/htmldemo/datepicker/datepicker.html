功能演示

DIV展示日历:
  • 日期控件展示
默认功能:
  • datepicker被绑定到一个form的input中。当输入框获得焦点时(点击,或使用tab键)可以打开　一个日历。选择一个日期后,点击页面上的其他地方,或按下Esc键关闭。如果选择一个日期后,输入的值就是你选择的日期。
显示其他月:
  • showOtherMonths: true,//显示其他月 selectOtherMonths: true//是否可选 设置以上两个属性可以看见其他月 并可以选择
显示今天/关闭:
  • showButtonPanel: true 设置这个属性 显示一个按钮用于选择今天的日期和完成按钮,关闭日历与布尔showButtonPanel选项。每个按钮是默认启用当栏显示,但可以关掉额外的选项。按钮的文本是可定制的。
显示时间选择:
  • 在 dtdatepicker 中 设置 setSelectDay:function(dd,mm,yy){alert("您选择了："+yy+"年"+(mm+1)+"月"+dd+"日");}函数 点击confirm按钮 完成带有时间的录入
下拉年月:
  • 设置 参数 changeMonth: true, changeYear: true 显示下拉列表方式的月和年来代替静态月/年头便于导航选择跨度较大时间。添加布尔changeMonth和changeYear选项即可
显示多个月:
  • 设置numberOfMonths属性为数字为2或更大的整数可以显示多个月在单个datepicker中。
限制可选:
  • minDate: -20,maxDate: "+1M +10D" 通过设置这两个属性 可以限制一些 时间不可点击
时间范围: To
  • 选择日期范围搜索
星期序列:
  • showWeek: true 设置 这个属性 可以让你看到时间是以按照周的方式来显示的。默认的计算遵循ISO 8601定义:这个星期从星期一开始的,这一年的第一周周四包含第一年的。这意味着一些天从一年可能被放置到周的归属感”一年。
